1. Setting Kecepatan Kamera

Setting di range antara 400-800 (opsional). Jika masih belum bisa, maka bisa diluar range tersebut.

2. Setting Focus Kamera

Setting fokus kamera dapat dilakukan secara otomatis. Jika kurang fokus, bisa di-adjust manual.

3. Setting Kecerahan Cahaya

Klik Screen Adjust dan atur bagian Adjustment. Semakin besar angka, semakin terang cahaya.

4. Mirror Hasil Kamera

Panah atas/bawah: membalik arah vertikal. Panah kanan/kiri: membalik arah horizontal.

5. Shape Search Setting III

Digunakan untuk menstabilkan batch number dan mendeteksi label.

Langkah: Atur kotak Model → Region (diluar kotak model) → Measurement → Klik Measure.

6. Barcode Setting

Klik ikon Barcode → Edit Region → Atur bentuk kotak → Apply → OK.

Lanjut ke Measurement: Teaching → Code Type = JAN/EAN/UPC → Ceklis Display of Decoded → Atur size angka.

7. OCR Setting

Klik ikon OCR → Edit Region → Pilih persegi → Measurement → Teaching menggunakan simbol # (angka) dan $ (huruf).

Masuk ke Judge Condition → ceklis OCV → lanjut Dictionary Setting & Regist.

8. 2D Code Setting

Klik ikon 2D Code → Setting Region → Measurement: Read Mode = DPM → Teaching → Display of Decoded → Atur Size.

9. Trend Monitor Setting

Menampilkan output berupa count NG dan OK. Klik Trend Monitor → Measurement → Tambahkan format seperti Barcode, OCR1-3.

Atur Judgement Range dan Display sesuai jumlah format yang dibaca.

10. Troubleshooting

  • Produk Terbaca NG: Cek format yang NG dan pastikan sesuai standar.
  • Camera Tidak Stabil: Ubah shutter speed atau periksa setting Shape Search III.
  • Capture Tidak Pas: Pastikan sensor pas dan kamera sejajar.
  • Batch Number Tidak Stabil/Terbacanya Buruk: Atur fokus dan pencahayaan kamera.
  • Barcode Salah: Periksa Verification String, pencahayaan, dan Code Type.
