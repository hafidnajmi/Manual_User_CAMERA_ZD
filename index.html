User Manual Kamera Omron FHV7

1. Setting Kecepatan Kamera

Setting di range antara 400-800 (opsional). Jika masih belum bisa, maka bisa diluar range tersebut.

2. Setting Focus Kamera

Setting fokus kamera dapat dilakukan secara otomatis.

Catatan: Jika masih kurang fokus, bisa di-adjust secara manual sampai terlihat fokus.

3. Setting Kecerahan Cahaya

Klik Screen Adjust, kemudian atur cahaya di bagian Adjustment.

Catatan: Semakin besar angka, semakin terang cahaya. Lampu memiliki 4 sisi.

4. Mirror Hasil Kamera

Gunakan tombol panah atas & bawah untuk membalik arah vertikal, kanan & kiri untuk arah horizontal.

5. Shape Search Setting III

Fungsi: Menstabilkan Batch Number, mendeteksi label (ada/tidak ada).

Langkah: Atur kotak Model sesuai objek, lanjut ke Region (buat kotak lebih besar), lalu Measurement. Klik "Measure".
