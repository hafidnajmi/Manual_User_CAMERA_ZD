1. Camera Image Input FHV Setup

Pengaturan dasar kualitas citra: kecepatan rana, fokus, pencahayaan, dan orientasi gambar.

  • Shutter Speed: Sesuaikan dengan kecepatan line (100‚Äì400 untuk sachet, 600‚Äì900 untuk conveyor).

3. Barcode Setting Reading

  1. Klik ikon Barcode.
  2. Masuk ke Region Setting ‚Üí klik Edit ‚Üí pilih bentuk persegi untuk area.
  3. Posisikan region meng-cover barcode dengan presisi ‚Üí klik Apply ‚Üí OK.
  4. Masuk ke Measurement:
    • Teaching: Code Type = JAN/EAN/UPC.
    • Display of Decoded: centang untuk menampilkan hasil pembacaan.
    • Size: ukuran tampilan angka (standar 50).
  5. Isi kolom sesuai kode barcode yang berjalan di mesin.

4. OCR Setting Text

  1. Klik ikon OCR ‚Üí Edit Region ‚Üí buat kotak area teks.
  2. Masuk ke Measurement ‚Üí Teaching.
  3. Gunakan simbol # untuk angka dan $ untuk huruf.
  4. Masuk ke Judge Condition ‚Üí centang OCV & isi huruf/angka sesuai pada produk.
  5. Buka Dictionary Setting ‚Üí pilih Dictionary Unit (OCR User Dictionary) yang diinginkan.
  6. Masuk Dictionary Regist ‚Üí klik Measure ‚Üí Character Regist untuk mendaftarkan karakter ke OCR User Dictionary.

8. Backround Supression Setting Enhancement

Mengurangi gangguan dari latar (glare/tekstur) yang mempengaruhi hasil inspeksi.

  3. Sesuaikan level filter sesuai kondisi cahaya.
  4. Gunakan terutama saat kemasan mengilap/pantulan kuat.

11. Troubleshooting Support

  • Produk terbaca NG: periksa setting format & pencahayaan.
  • Kamera tidak stabil: sesuaikan shutter speed atau konfigurasi Shape Search.
  • Capture tidak pas: cek sensor trigger & posisi kamera.
  • Batch number tidak stabil: periksa fokus & pengaturan lampu.
  • Barcode salah: periksa Code Type, pencahayaan, dan verification string.
